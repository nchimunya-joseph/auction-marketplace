About

This project was inspired by my personal experiences and passion for ebay and Auction.com. It all started as a simple project with no GUI, just working with a plain console application. As a Portfolio Project for Holberton School, I had the opportunity to bring this idea to life and showcase my skills and creativity.

Key Features

Auction Management

Allow users to manage their own auctions, including editing the auction details, extending the end date/time if desired, and closing the auction once it has ended.

Auction Listing

Provide a platform for users to create and list their items for auction. Users should be able to specify details such as the title, description, starting bid, and end date/time for each auction.

Watchlist

Allow users to add auctions to their watchlist, making it easier for them to keep track of auctions they are interested in without necessarily placing a bid.
